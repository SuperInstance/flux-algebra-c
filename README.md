# flux-algebra-c

C99 port of the [flux-algebra](https://github.com/SuperInstance/flux-algebra) Python library — HarmonicRing operations, group theory for harmony, and tuning field arithmetic.

## What It Does

- **HarmonicRing** — Z/nZ for pitch classes (default 12-TET): addition (transposition), multiplication (interval composition), inversion, ideals
- **IntervalRing** — just-intonation frequency ratios as numerator/denominator pairs: add intervals (multiply ratios), subtract, invert, convert to cents
- **Group theory** — GCD, modular inverse, element order, subgroup membership
- **Interval vectors** — IC distribution for pitch-class sets
- **Prime form** — Fortean prime form computation
- **Ideal structure** — all ideals of Z/nZ, ideal generated by chord tones

The ideal structure of Z/12Z yields musically meaningful objects:
- `{0}` — unison
- `{0, 6}` — tritone pair
- `{0, 4, 8}` — augmented triad
- `{0, 3, 6, 9}` — diminished seventh chord
- `{0, 2, 4, 6, 8, 10}` — whole-tone scale
- `Z/12Z` — chromatic scale

## C API

```c
#include <flux_algebra.h>

/* Pitch-class ring operations */
FaHarmonicRing hr;
fa_harmonic_ring_init(&hr, 12);
int g_plus_e = fa_add(&hr, 7, 5);  /* 0 (C) */
int ic = fa_interval_class(&hr, 0, 7);  /* 5 */

/* Units of Z/12Z */
int units[12];
int n = fa_units(&hr, units, 12);  /* {1, 5, 7, 11} */

/* Ideal generated by a chord */
int gens[] = {0, 4, 7};  /* C major */
FaIdeal ideal = fa_ideal_generated_by(&hr, gens, 3);

/* Interval vector */
int counts[7];
int pcs[] = {0, 4, 7};
fa_interval_vector(&hr, pcs, 3, counts);

/* Just-intonation: P5 + M3 = M7 */
FaIntervalRing ir;
fa_interval_ring_default(&ir);
int64_t num, den;
fa_ir_add(&ir, 3, 2, 5, 4, &num, &den);  /* 15/8 */
double cents = fa_ir_cents(num, den);  /* ~1088 cents */
```

## Build

```sh
make        # builds libflux_algebra.a
make test   # builds and runs tests
make clean
```

No dependencies beyond C99, `libc`, and `libm`.
